# Macの環境構築スクリプト
最初に実行して環境を構築するためのスクリプトです。
Mac OSで最初にターミナルから実行してください。

```
$ chmod +x setup.sh
$ ./setup.sh
```

以下に自分の開発環境のメモを置いておきます。
